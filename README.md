## 🏡 このアプリについて

急速に高齢化が進むおおくら地区では、民生委員が担当を決めて単身高齢者の見守りを行っています。しかし民生委員自身の高齢化により、実際に対象者のお宅へ足を運ぶ現在の状況が次第に困難になっています。

そこで、毎日の安否確認をアプリで行い、必要な時にすぐ声掛けができるシステムを作りました。

プログラミング未経験の状態からAIの力を借りて開発しました。

---

## ✨ 主な機能

| ボタン | 機能 |
|---|---|
| 🌅 おはよう！ | 朝の安否確認・音声メッセージ再生・クラウド記録 |
| 🌙 おやすみなさい | 就寝前確認・音声メッセージ再生・クラウド記録 |
| 💬 ご相談 | 担当者へメール通知を即時送信 |

---

## 📊 記録の仕組み

```
対象者がボタンを押す
　↓
Googleスプレッドシートに自動記録
　↓
担当者がいつでも確認できる
```

### スプレッドシートの構成

| シート名 | 内容 |
|---|---|
| 記録シート | 毎日のボタン操作の記録 |
| 担当者シート | 担当者名・役割・メールアドレス |
| 利用者シート | 利用者名・担当者の対応表 |
| 今日の状況 | 全員の当日状況を一覧表示 |

## 🚀 導入手順

### 担当者の設定
1. Googleスプレッドシートの「担当者シート」に名前・役割・メールアドレスを入力
2. アプリの担当者リストに自動反映されます

### 利用者のスマホへの導入
1. SafariまたはChromeでアプリのURLを開く
2. セットアップ画面でお名前・アイコン・担当者を設定
3. ホーム画面にショートカットを追加
